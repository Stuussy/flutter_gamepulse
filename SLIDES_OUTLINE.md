## СЛАЙД 9: Алгоритмы
```
📐 РАСЧЕТ FPS:
1. Performance score CPU/GPU
2. Сравнение с требованиями игры
3. Множители RAM и соответствия
4. Итоговый расчетный FPS

🔍 BOTTLENECK DETECTION:
if (RAM < 16GB) → апгрейд RAM
else if (CPU < GPU × 0.7) → апгрейд CPU
else → апгрейд GPU

🤖 AI FALLBACK:
• "стоит ли" → цена/качество
• "зачем" → необходимость
• "FPS" → конкретные цифры
• "плюсы" → преимущества
```

## ДОПОЛНИТЕЛЬНЫЕ СЛАЙДЫ (для вопросов)

### Слайд А: Детали базы данных
```
MongoDB Schema:
{
  username: String,
  email: String (unique),
  password: String (bcrypt hash),
  pcSpecs: {
    cpu: String,
    gpu: String,
    ram: String,
    storage: String,
    os: String
  }
}

Индексы: email (unique)
Связи: One-to-One (User ↔ PC Specs)
```

### Слайд Б: API Endpoints
```
AUTH:
POST /register
POST /login
POST /forgot-password

USER:
POST /add-pc
POST /user-profile

GAMES:
POST /check-game
POST /performance-graph

AI:
POST /ai-upgrade-recommendations
POST /ai-upgrade-explanation
```

### Слайд В: Расчет производительности
```
Performance Score Formula:
━━━━━━━━━━━━━━━━━━━━━━━━━
CPU: Intel i5-12400 = 150
GPU: RTX 3060 = 220
RAM: 16GB = base multiplier

FPS = base_fps × (cpu_score/100)
             × (gpu_score/100)
             × ram_multiplier

Пример: 60 × 1.5 × 2.2 × 1.0 = 198 FPS
```
